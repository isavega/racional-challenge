## Racional Challenge

Repositorio con dos proyectos:

- **`racional-app/`**: dashboard de inversiones (React + TypeScript + Vite, Firestore realtime, Chart.js, Tailwind).
- **`racional-api/`**: API de inversiones (NestJS + TypeScript + Firestore) con Swagger.
